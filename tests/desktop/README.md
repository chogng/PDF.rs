# Desktop integration fixtures

`readable-preview.pdf` is the deterministic `m4.readable-preview.v1` visual
acceptance document. It is generated by:

```sh
cargo run --locked --package pdf-rs-generate -- \
  --readable-preview tests/desktop/readable-preview.pdf
```

The two-page Letter document, its block TrueType outlines, content streams,
and colors are entirely project-authored. It is registered as
`LicenseRef-PDF.rs-SelfAuthored-Test` and redistributable with this repository.
The generator replay test requires byte-for-byte identity.
